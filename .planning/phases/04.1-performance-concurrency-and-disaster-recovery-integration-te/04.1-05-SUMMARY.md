## Decisions Made

- **Option A for Postgres override:** `@DynamicPropertySource` in `LargeDbStressIT` registers `spring.datasource.url/username/password` from `seededPostgres::getJdbcUrl` etc. Spring's dynamic property registry wins over the `@ServiceConnection`-backed `postgresContainer` bean from `TestcontainersConfiguration`, so the seeded container becomes the datasource without needing a new `@TestConfiguration` class. Option B (dedicated `SeededStressContainerHolder`) was not needed — no Spring conflict observed.

- **Static init block for container bootstrap:** The seeded container must be started before `@DynamicPropertySource` fires. Using a `static final` field initialized via a `static {}` block guarantees the container is up before any Spring context property resolution.

- **Relative Dockerfile path:** `Paths.get("engine-service/src/test/docker/perf-db").toAbsolutePath()` resolves from Gradle's CWD (the repository root) regardless of how the test is invoked.

- **Conservation invariant scope:** The `stress_accounts` rows are seeded once at container init time and never modified by the REST probes (the engine's debit endpoint operates on engine-managed accounts, not `stress-acc-*` rows). Therefore `initialTokensTotal == totalFinalBalance` (delta == 0) is the correct conservation signal.

## End-to-End Verification

To run the full Phase 04.1 suite after merging all worktree branches:

```
.\gradlew.bat :engine-service:performanceTest
.\gradlew.bat :engine-service:disasterRecoveryTest
```

`LargeDbStressIT` runs under `performanceTest` (tagged `performance`). On first run for a given `(accounts, transactionsPerAccount)` pair, the Docker image build may take several minutes. On cache hit (second run), the build step is skipped and startup is near-instant.

For CI with time constraints, override the defaults:
```
.\gradlew.bat :engine-service:performanceTest -Dcertacota.perf.stress.accounts=500 -Dcertacota.perf.stress.transactions-per-account=100
```

This uses image `certacota-perf-db-a500-t100` which seeds quickly.
